MMX Randomizer

GUI that sends a randomized boss order of game for user to try out different combinations (For context the original games allowed players to choose whichever levels they wanted; naturally there would be an optimal sequence of stages to choose which will give the easiest time possible)

The intent of this randomizer is to give players a sub-optimal route to take which effectively changes the way the games would be played (more sub-optimal routes leads to interesting challenges) Also extra constraints for the player too

Also allows to email the user the text of the randomized boss order (This is a small project I made when I was new to coding)

Developed in Python using the Tkinter GUI module

Bear Maps

Class Project for Data Structures Course. Bear Maps functions similar to Google Maps but downscaled to be just within the UC Berkeley campus vicinity.

Utilizes Java 7 Standard Library datastructures such as Hash Maps, ArrayLists, etc. Also created Quad-Tree Data structure to specifically allow for zooming in/out of the map. Implemented A* algorithm to find best routes.

Developed in Java with XML to parse data of location points in Berkeley, CA.
